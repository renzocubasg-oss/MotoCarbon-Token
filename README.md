# MotoCarbon Token

## Proyecto Final - Hackathon Carbon Tokenization

**Universidad Nacional Mayor de San Marcos (UNMSM)**

### Descripción

MotoCarbon Token es una solución basada en Blockchain que registra el CO₂ evitado por mototaxis eléctricas y genera tokens ambientales para incentivar la movilidad sostenible.

### Tecnologías utilizadas

- Python
- Pandas
- Matplotlib
- Solidity
- Blockchain
- GitHub

- docs/
# MotoCarbon Token

Proyecto desarrollado para la Hackathon de Tokenización de Carbono.

## Descripción

MotoCarbon Token propone un sistema basado en Blockchain para registrar el CO₂ evitado por mototaxis eléctricas y recompensar a los conductores mediante tokens ambientales.

## Tecnologías utilizadas

- Python
- Pandas
- Matplotlib
- Solidity
- GitHub
